# FABULA

## Description ##
A little Python calculator based on postfix notation. 

## HOW TO USE:
Type **whereami** if you need help.

## HOW TO READ POSTFIX NOTATION:
### Example 1
This expression:

```
3 18 + 2 **
```

Is equivalent to:

```
(3 + 18)^2 = 441
```
### Example 2

And this expression:

```
7 1 + 5 3 + *
```
Is the same as:

```
(7 + 1)*(5 + 3) = 64
```
